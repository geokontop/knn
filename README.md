# Usage

### Node
If you have [node](http://nodejs.org/), you can install `brain.js` with [npm](http://npmjs.org):

```
npm install gk-knn.js
```

Or if you prefer yarn:
```
yarn add gk-knn.js
```

At present, the published version of gk-knn.js is approximately 1.0.0.
